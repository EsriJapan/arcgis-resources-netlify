管理者の操作

ArcGIS Pro のライセンスとエクステンションは ArcGIS Online で管理されるため、ArcGIS Online の利用登録が必要です。
ArcGIS Online の登録とライセンスの割り当ては、複数のステップで構成されています。

ArcGIS Online の組織サイトを既にご利用の方は「メンバーの追加」からライセンスの割り当てを行ってください。
各ステップの詳しい手順については、各ステップのタイトルからリンクしている ArcGIS Online スタートアップ ガイドをご参照ください。

1. サブスクリプションの有効化

「サブスクリプションの有効化」とは、ArcGIS Online のクラウドに参加するための手続きに相当します。
ご契約時に受信されたメールまたはライセンス コンファメーションに記載されている URL から <サブスクリプション ID 有効化ページ> にアクセスし、アカウントの設定を行います。

保守契約が有効な ArcGIS Desktop ライセンスをお持ちで、サブスクリプションの有効化 URL についての案内メールを受け取ったことがない、ArcGIS Online の利用申請を行ったことがない場合は、利用申請 を行ってください。

2. 組織の設定

ArcGIS Online では組織専用のサイトを作成することができます。
組織サイトの設定ページで、ショートネーム (https://○○.maps.arcgis.com の○○の URL 名を決定)、言語、地域、セキュリティを設定します。

3. メンバーの追加

組織サイトの構成を設定後、管理者は ArcGIS Pro を使用したい利用者を組織サイトに招待できます。
組織サイトから、メンバーの [名]、[姓]、[電子メール アドレス]、サイン イン時に使用する [ユーザー名]、[ユーザー タイプ]、[ロール]、[一時パスワード] を設定します。

4. ライセンスの割り当てと解除

ArcGIS Pro で使用するライセンス (エディションおよびエクステンション) は、ArcGIS Online 組織ページより割り当てを行う必要があります。
管理者の方は組織サイトから各メンバーにライセンスの割り当てを行ってください。

ステップ 3 において、組織サイトへのメンバーの追加が確実に行われている必要があります。

ステップ 3 において、Creator、Professional、Professional Plus ユーザー タイプを設定している場合はライセンスの割り当ての必要はありません。

5. 組織サイトへ加入

ArcGIS Pro を利用するためには、ArcGIS Online の組織サイトのメンバーである必要があります。
ArcGIS Online の組織サイトの管理者から招待を受けたメンバーは、メールに記載されている URL から組織サイトへアクセスし、組織サイトに加入してください。


ArcGIS Desktop ライセンスとして購入している際の ArcGIS Pro を利用するための申請について

保守契約が有効な ArcGIS Desktop ライセンスをお持ちの場合は、ArcGIS Pro をご利用いただけます (ArcGIS Desktop 1 ライセンスにつき: 1 ユーザー (Creator)、1 ArcGIS Pro ライセンス)。

組織内で ArcGIS Online がまだ一度も申請されていない場合、ArcGIS Pro を利用する前にライセンスの割り当てに必要な ArcGIS Online (ArcGIS Desktop ユーザー向け) の利用申請をしていただく必要があります。

以下の手順で ArcGIS Online の利用申請を行ってください。

  1. ArcGIS Online (ArcGIS Desktop ユーザー向け) お申込みページから「個人情報の取り扱い」および「ArcGIS Online (ArcGIS Desktop ユーザー向け) 利用条件」をご確認の上、申込者情報入力ページに必要事項を入力し、[次へ] をクリックします。
  2. 申し込み内容をご確認の上、[申込] をクリックします。申し込み受付完了の通知メールが、入力されたメール アドレス宛に送信されます。

弊社にてお客様の保守契約状況を確認後、3 ~ 4 営業日以内に「【ArcGIS Online サブスクリプション ID 発行のご案内 (ArcGIS Desktop ユーザー向け)】」というメールをお送りいたします。メールを受信しましたら、ステップ 1「サブスクリプションの有効化」を行ってください。
